# 文件搜索工具

一款基于 PyQt5 的 Windows 桌面文件搜索工具，支持按文件名和文件内容进行高速搜索，并提供文件预览、文件操作等丰富功能。

## 功能特性

### 文件名搜索

- **多种匹配模式**：支持通配符匹配（如 `*.txt`）、绝对匹配、正则表达式
- **大小写敏感**：可切换是否区分大小写
- **多种搜索算法**：
  - 广度优先（BFS）
  - 深度优先（DFS）
  - 文件优先
  - 文件夹优先
  - 按扩展名分组
  - 按大小排序
  - 按修改时间排序
- **搜索范围**：可选择仅搜索文件、仅搜索文件夹、或全部

### 内容搜索

- **多种匹配模式**：支持普通文本、绝对匹配、正则表达式、多行模式
- **大小写敏感**：可切换是否区分大小写
- **并发搜索**：支持多线程加速内容搜索，可自定义线程数（1-16）
- **预览上下文**：可设置匹配行上下文显示行数（1-50 行）

### 文件过滤

- **实时过滤**：在搜索结果中输入关键词即时过滤
- **过滤范围**：可按全部、文件名、路径、后缀进行过滤
- **正则过滤**：支持使用正则表达式进行过滤

### 文件预览

- **内容预览面板**：选中文件后右侧即时显示内容
- **匹配高亮**：搜索关键词在预览中以高亮样式标出
- **上下文显示**：匹配行周围显示指定行数的上下文
- **分页浏览**：大文件支持分页浏览，每页 500 行
- **完整内容模式**：可切换查看文件完整内容
- **字符统计**：显示预览内容的字符数

### 文件操作（右键菜单）

- 打开文件
- 用资源管理器打开所在文件夹
- 复制文件到指定目录
- 复制完整路径到剪贴板
- 复制匹配摘要（包含文件路径、行号和匹配内容）
- 重命名
- 删除到回收站（需安装 send2trash）
- 彻底删除（Shift+Delete）
- 查看文件属性
- 按名称排序（A-Z / Z-A）
- 刷新结果（移除已不存在的文件）

### 文件属性对话框

- **常规标签页**：显示文件名、类型、位置、完整路径、大小、创建/修改/访问时间、文件属性（只读、隐藏、系统、存档等）
- **内容标签页**：文件内容预览，支持"加载更多"和"加载完整文件"
- **快捷操作**：打开文件、打开所在文件夹、复制完整路径

### 界面特性

- **深色/浅色主题**：自动检测系统主题，支持一键切换，切换时带有平滑过渡动画
- **自定义标题栏**：无边框窗口设计，支持拖拽移动、双击最大化、右键系统菜单
- **窗口边框缩放**：通过原生事件实现窗口边缘拖拽调整大小
- **动画效果**：按钮点击缩放动画、进度条条纹动画、表格淡入动画、状态栏闪烁提示
- **拖拽选择目录**：支持将文件夹直接拖拽到搜索目录输入框
- **中文右键菜单**：所有文本输入框的右键菜单均已汉化
- **设置持久化**：自动保存和恢复所有搜索选项、主题设置等
- **管理员权限**：支持以管理员身份重启，以便访问受保护的文件

### 其他选项

- **扩展名白名单**：仅搜索指定扩展名的文件（如 `.txt,.py`）
- **扩展名黑名单**：排除指定扩展名的文件（如 `.exe,.dll`）
- **排除常见垃圾目录**：自动跳过 `.git`、`node_modules`、`__pycache__`、`venv` 等目录

## 使用方法

### 启动

```bash
python file_search_tool.py
```

### 基本操作流程

1. **选择搜索目录**：点击"选择"按钮或直接拖拽文件夹到输入框
2. **文件名搜索**：在"文件名搜索"区域输入匹配模式，选择搜索算法，点击"搜文件"、"搜文件夹"或"全部"
3. **内容搜索**：先完成文件名搜索，然后在"内容搜索"区域输入搜索内容，点击"搜索内容"
4. **查看结果**：在中间表格中查看搜索结果，点击行可在右侧预览文件内容
5. **过滤结果**：在顶部过滤栏中输入关键词实时过滤搜索结果
6. **文件操作**：右键点击结果行可进行打开、复制、删除等操作

### 快捷键

- **Esc**：取消正在进行的搜索

## 依赖说明

### 必需依赖

- **Python 3.6+**
- **PyQt5**：GUI 框架

```bash
pip install PyQt5
```

### 可选依赖

- **send2trash**：用于将文件移动到回收站（如未安装，删除到回收站功能将不可用，但仍可使用彻底删除）

```bash
pip install send2trash
```

## 支持的文本文件类型

内容搜索支持以下扩展名的文本文件：

```
.txt .py .json .xml .html .htm .js .css .md .log .ini .cfg .yaml .yml
.csv .sh .bat .ps1 .c .cpp .h .hpp .java .go .rs .swift .kt .sql .r
.rmd .tex .markdown .rst .vue .jsx .tsx .scss .sass .less .toml
.dockerfile .makefile .cmake
```

## 自动排除的目录

开启"排除常见垃圾目录"后，搜索时将自动跳过以下目录：

```
.git .svn .hg node_modules __pycache__ .idea .vscode .vs dist build
target bin obj .gradle .mvn .cache .tox .mypy_cache .pytest_cache
venv .venv env .env .direnv vendor Pods .next .nuxt .svelte-kit
coverage .coverage __pypackages__ .pytype htmlcov .sass-cache
```

## 平台说明

本工具专为 **Windows** 平台设计，使用了以下 Windows 特有功能：

- `os.startfile()` 打开文件
- `ctypes.windll` 检测管理员权限和提权重启
- `winreg` 读取系统主题设置
- 原生窗口事件处理边框缩放
